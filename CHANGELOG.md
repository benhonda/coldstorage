- fix(security): **cross-account leak ‚Äî the daemon's journal was machine-wide; state is now per-user (`UserSession`).** The journal was a single `~/Library/Application Support/ColdStorage/coldstore.sqlite` with no owner column, and sign-out dropped only the AWS credentials + the MasterKey ‚Äî so signing out of account A and into account B on the same Mac showed B **all of A's files, folder paths, sizes and watched-folder registry**, which the run loop then kept scanning under B's session (new files in them would have been archived into B's vault).
